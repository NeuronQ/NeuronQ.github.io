## Requirements

Nodejs global deps:

```
npm install -g concurrently
npm install -g lite-server
```

## How to tun local HTTP server and auto-recompile on change

```
npm start
```

### How to build

```
npm build
```

(alias of `gulp`)

### How to run local HTTP server

```
npm serve
```

(alias of `lite-server`)

### How to auto-rebuild on change

```
npm watch
```

(alias of `gulp watch`)
